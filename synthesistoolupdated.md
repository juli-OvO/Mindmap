# Synthesis Tool — CLAUDE.md (Updated)

## Project
Personal cognitive instrument:
Capture fragments → observe patterns → negotiate meaning → optional synthesis.

Core shift:
- Input is fast, raw (voice/text)
- Interpretation is slow, intentional (manual structuring)

This tool is not a productivity app.
It is a record of thinking forming over time.

---

## Input Modes (Primary Shift)

### Voice Input (PRIMARY)
Press → speak → release

- Speech-to-text
- Auto-extract fragments + tags
- Generates node clusters
- Visual: ink spreading outward during capture

### Text Dump (SECONDARY)
Paste or write large chunk

- Parsed into fragments
- Auto-tagged
- Treated same as voice session

Manual editing exists but is NOT primary workflow.

---

## Temporal System (NEW)

Instead of modifying a single graph:
→ The system accumulates sessions over time

Each input = a session layer

### Timeline (Right Side UI)
- Vertical timeline
- Each entry = one input session
- Hover/scrub:
  - nodes fade in/out
  - connections shift
  - particle flow reconfigures

Behavior:
- No overwrite
- Only accumulation + relation

User decides:
- merge
- connect
- isolate

---

## Core Interaction Model

INPUT (fast)
    ↓
AUTO PARSE (system suggests structure)
    ↓
USER INTERPRETATION (slow, manual)
    ↓
CONNECTION + MEANING
    ↓
OPTIONAL SYNTHESIS (text)

## Nodes (Updated)

- Not rigid circles → soft ink density shapes
- Opacity + spread reflects weight
- Tags embedded lightly inside
- Nodes emerge from ink, not placed mechanically

---

## Connections

Still 3 types:
- echoes (parallel flow)
- leads-to (directed flow)
- counter (turbulent collision)

BUT:
- rendered as flow fields, not lines
- directional ink movement instead of geometric clarity

## Voice Capture Visualization (NEW CORE MOMENT)

During speaking:
- ink expands from input point
- words condense into nodes
- pauses create gaps
- heavier words = darker density

This is NOT decoration:
It externalizes internal thinking flow.
