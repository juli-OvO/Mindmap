Terrain Border Mechanisms

How contour edges are drawn. Same node cluster, same palette — different border/rendering approach.

A — Soft Blob (current design)

Radial gradient arcs + ctx.filter blur. Borders dissolve organically. No hard edges anywhere.

B — Topographic Contour Lines

Thin strokes traced at each density threshold using marching squares. No fill — pure line drawing. Feels like a surveyor's map.

C — Filled Bands + Hairline Contours

Soft filled regions (like A) with thin contour strokes drawn on top at each threshold. Layered depth — blob fill + topo lines.

D — Inner Radiance (no border)

Single large radial gradient per node cluster — no discrete bands. Continuous light bloom from center. Most ambient, least structured.
